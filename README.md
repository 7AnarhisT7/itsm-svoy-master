# ITSM Svoy Master

ITSM система для сервисных служб сегмента HoReCa.

## О проекте

Система управления сервисными заявками для обслуживания оборудования и инфраструктуры в заведениях HoReCa.

### Основные модули
- Управление заявками
- Каталог услуг
- Управление клиентами
- SLA и дедлайны
- Учёт оборудования
- Уведомления
- Отчёты и аналитика

## Стек технологий

| Слой | Технология |
|------|-----------|
| Бэкенд | Python + FastAPI |
| База данных | PostgreSQL |
| Фронтенд | React |
| Контейнеры | Docker |
